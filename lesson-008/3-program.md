# 有理数及有理数加减乘除运算

有理数的类名用 Rational 表示.

## Req - 1

- 定义一个 Rational 类，传入两个整数，分别表示分子和分母；
- 定义一个方法，获取有理数的字符串表示形式

测试用例:

- (1, 2) -> 1/2
- (2, 1) -> 2

## Req - 2

- 如果分子和分母有公约数，则需要化简

测试用例:

- (3, 9) -> 1/3
- (3, 7) -> 3/7

## Req - 3

- 考虑分子、分母传入负数的情况

测试用例:

- (1, -2) -> -1/2
- (-1, -2) -> 1/2
- (-4, 2) -> 2

## Req - 4

- 分母为0的异常处理

测试用例:

- (1, 0) -> ZeroDivisionError

## Req - 5

- 入参为非整数的异常处理

测试用例:

- (1.0, 2) -> TypeError
- (1, 2.0) -> TypeError
- ("1", 2) -> TypeError
- ("1", "2") -> TypeError
- (1, "2") -> TypeError

## Req - 6

- 易用性考虑: 如果分母为1，则初始化的时候只需要传入一个分子即可。

测试用例:

- (1) -> 1
- (-2) -> -2
